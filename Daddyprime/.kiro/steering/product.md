# Product: Daddy

Daddy is a real-time, AI-powered social messaging web app. It's inspired by Telegram's UI and targets Nigerian users (phone auth uses +234 format).

## Core Features
- Real-time 1:1 and group chat via Supabase Realtime
- AI-powered action item detection from messages (Gemini)
- Chat summarization (Gemini)
- Triage inbox with Priority, Groups, and Actions tabs
- Pinned chats, message reactions, scheduled messages, soft-delete
- Voice/video call overlay (UI only)
- Dark/light mode with user preference persistence
- Phone number OTP auth (via Express backend) + Supabase email/social auth

## Target Users
Nigerian users; phone numbers validated against +234XXXXXXXXXX format.
