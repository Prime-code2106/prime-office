# Hooks Directory

This directory contains custom React hooks for the portfolio website.

## Suggested Custom Hooks

1. **useNavigation.ts** - Handle page navigation logic
2. **useScrollToSection.ts** - Smooth scrolling to page sections
3. **useContactForm.ts** - Contact form submission logic
4. **useLocalStorage.ts** - Local storage operations
5. **useTheme.ts** - Dark/light theme management

## Current Implementation
Currently, navigation and other logic is handled directly in components. 
As the application grows, consider extracting common logic into custom hooks for better reusability.
