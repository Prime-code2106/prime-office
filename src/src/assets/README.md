# Assets Directory

This directory contains all static assets for the portfolio website:

- **images/**: Store all image files (photos, backgrounds, project screenshots)
- **fonts/**: Custom fonts if needed
- **icons/**: SVG icons and custom icon files
- **documents/**: PDFs and other document files

## Current Assets
- Resume: `/public/resume/Adewale_Samuel_Prime_Resume.pdf`
- Background images: Using Unsplash API in components
- Icons: Using Lucide React icon library

## Notes
- Images are currently sourced from Unsplash API
- To replace with local images, save them in `/src/assets/images/` and update component imports
